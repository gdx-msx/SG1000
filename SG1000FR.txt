-------------------------------------------------
  Lanceur de ROM SG-1000(II), FG-1000, SC-3000, 
 Othello Multivision & Mark III/Gam*boy coréenne
      pour MSX avec une cartouche Franky,
      Musical Memory Mapper ou Playsoniq

                v0.96 par GDX
-------------------------------------------------

SG1000.COM permet de jouer aux jeux SG-1000 sur MSX. Utilisez une cartouche
Franky, Musical Memory Mapper ou Playsoniq pour obtenir le son comme avec la
console originale. Autrement, vous n'aurez aucun son.


Installation:

 Sous MSX-DOS1, copiez les fichiers SG1000.COM, SG1000.DAT et SG1000F.DAT sur
 le même disque où se trouvent vos Roms.

 Sous MSX-DOS2 ou Nextor, vous pouvez copier SG1000.COM dans un autre dossier
 ou disque si le chemin est spécifié dans la variable d'environnement PATH.
 
 SG1000F.DAT n'est pas nécessaire si vous n'avez pas de Franky.
 

Utilisation:

 SG1000.COM est une commande MSX-DOS1/2. 
 Entrée de la commande et le nom du fichier de la ROM à la suite.

  >SG1000 RomName.ext

 - L'option /F doit être ajoutée si vous utilisez la cartouche Franky.
   Autrement, vous pouvez renommer SG1000F.DAT par SG1000.DAT pour éviter de
   spécifier l'option.
 - L'option /S peut être utilisée pour obtenir des couleurs proche de la
   Sega Master System.


Exemple :

  A>SG1000 DROL.SG

Notes:
- Les Roms qui nécessitent une extension de RAM utilisent les modes d'écran
  de la SG-1000. Certains d'entre eux ont été vendus pour la Gam*boy ou
  l'Aaronix MarkIII (version coréenne ou taïwanaise de la MarkIII). Plusieurs
  de ces jeux ont un son mal converti d'origine.
- Le clavier n'est pas pris en charge.
- Cette version prend en charge les ROM jusqu'à 48 Ko.
- Musical Memory Mapper est une cartouche faite par Popolon (fr).
   https://www.msx.org/wiki/Popolon_Musical_Memory_Mapper
   http://www.ebsoft.fr/shop/fr/home/35-musical-memory-mapper.html
- Franky & Playsoniq sont des cartouches faites par SuperSoniqs.
   http://supersoniqs.com/category/playsoniq/
